# Renderer Guidance Card Alignment Design

## Goal

Align renderer helper cards, action headers, and number-pad supporting text so the lesson interaction area reads as one guidance system.

## Scope

- Introduce a shared renderer guidance card component.
- Use it for:
  - helper/context blocks
  - action header blocks
- Bring number-pad supporting text onto the same typography rhythm.

## Expected Result

The renderer area should feel visually coordinated instead of mixing separate helper and action card treatments.
