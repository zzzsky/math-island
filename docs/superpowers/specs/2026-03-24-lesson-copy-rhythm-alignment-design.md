# Lesson Copy Rhythm Alignment Design

## Goal

Align lesson copy rhythm so the left status cards, right feedback banner, and renderer action guidance all use the same short, direct structure.

## Scope

- Keep lesson progression unchanged.
- Shorten and align the wording for:
  - renderer action headers
  - timed lesson initial warning
- Reuse the same concise style already used by the lesson status cards.

## Copy Rhythm

- short title
- one sentence body
- direct instruction or direct result

Examples:

- `准备作答 / 先看题目，再提交这次答案。`
- `正在重试 / 先看提示，再判断一次。`
- `已确认 / 答案已确认，马上进入下一题。`

## Expected Result

The lesson screen should read like one coordinated teaching surface instead of multiple subsystems with different phrasing styles.
