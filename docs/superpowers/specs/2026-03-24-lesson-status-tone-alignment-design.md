# Lesson Status Tone Alignment Design

## Goal

Align lesson-screen status emphasis so the left status cards and right answer feedback banner use the same tone language.

## Scope

- Add a shared lesson status tone model.
- Keep answer evaluation and lesson progression unchanged.
- Use the shared tone model for:
  - left attempt status card
  - left timer pressure card
  - right answer feedback banner

## Tones

- `Neutral`
- `Highlight`
- `Retry`
- `Confirmed`
- `Warning`

## Expected Result

The lesson screen should feel like one coordinated state surface instead of separate left/right feedback systems.
