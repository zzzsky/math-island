# Math Island 手测 Checklist（Android Tablet）

> 适用版本：`0.1.0` 候选版  
> 基线提交：`a6a695a`  
> 更新时间：2026-03-31

## 1. 测试准备

- [ ] 使用平板模拟器，横屏启动（建议 `1280x800`，API 33+）
- [ ] 安装最新 Debug 包并确认可正常打开
- [ ] 清理历史进度（确保可验证首轮引导、解锁与奖励链路）
- [ ] 记录本轮测试构建信息（分支、commit、时间）

## 2. P0 冒烟主链路（必须通过）

### 2.1 首页 Home

- [ ] 首页正常加载，无白屏/闪退
- [ ] `继续冒险` 可进入上次进度对应流程
- [ ] `打开地图` 可进入地图页
- [ ] `打开宝箱` 可进入宝箱页
- [ ] `家长入口` 可进入家长验证页

### 2.2 地图 Map + 岛屿面板

- [ ] 地图加载成功，岛屿与路径可见
- [ ] 已解锁/未解锁/已完成状态显示正确
- [ ] 点击岛屿可打开 Overlay 面板
- [ ] 面板内课程卡 `开始` 可进入关卡（稳定合同：`panel-start-<lessonId>`）
- [ ] 地图回流摘要、左侧岛屿列表、右侧 handoff 文案保持一致
- [ ] 新岛 / 宝箱 / 回放 / 继续推进 4 类回流 CTA 与强调样式正确

### 2.3 关卡 Level（答题流程）

- [ ] 题干与选项/输入区渲染正常
- [ ] 未作答时提交按钮状态正确（禁用或不可继续）
- [ ] 正确作答反馈正确，可继续下一题
- [ ] 错误作答反馈正确，可重试/继续逻辑正确
- [ ] 限时课超时后进入统一 timeout 语义，不出现卡死或错误继续提交
- [ ] 左侧 support rail、右侧反馈 banner、题面 prompt 的状态语言一致
- [ ] 关卡结束可进入奖励弹层

### 2.4 奖励 Reward + 宝箱 Chest

- [ ] 奖励弹层信息正确（奖励、按钮、文案）
- [ ] staged reveal 节奏正确，不出现核心内容缺失或闪烁
- [ ] 结果舞台（kind pill / spotlight / detail / action）层级完整
- [ ] 奖励后进度已更新到地图/首页
- [ ] 宝箱页面可展示贴纸或空态
- [ ] 宝箱返回地图按钮可用（`chest-open-map`）

### 2.5 家长验证 + 家长总结

- [ ] 家长验证页可正常作答（`parent-answer-<value>`）
- [ ] 验证通过后可进入家长总结页
- [ ] 总结页关键数据展示正常（待复习/进度/表现）
- [ ] 返回主流程后状态一致

## 3. P1 核心功能细测

### 3.1 题型覆盖

- [ ] 单选题（choice）
- [ ] 数字键盘题（number-pad）
- [ ] 尺子题（ruler）
- [ ] 跟读/口令题（chant）
- [ ] 分组题（grouping）
- [ ] 排序题（sorting）

### 3.2 Challenge Island 专项

- [ ] 混合题跑通（mixed run）
- [ ] 计时冲刺（timed sprint）计时与判定正确
- [ ] 完成后评级展示正确（grading）
- [ ] 支持 replay 与 retry，且流程闭环正确

### 3.3 导航与状态保持

- [ ] 首页/地图/关卡/宝箱/家长页间跳转无错乱
- [ ] 系统返回键行为符合预期
- [ ] App 切后台再回前台，页面与进度不异常
- [ ] 重启 App 后进度持久化正确（DataStore）

### 3.4 外围页面设备侧检查

- [ ] Home 的 hero、推荐卡、操作列在 `1280x800` 下无重叠
- [ ] Chest 页面滚动正常，贴纸网格换行正确
- [ ] Parent Gate 答案区自动换行，Parent Summary 内部滚动正常

## 4. P2 稳定性与体验

- [ ] 连续快速点击关键按钮，不出现重复提交/卡死
- [ ] 低网或无网（若资源全本地）不影响核心流程
- [ ] 长时游玩（15-30 分钟）无明显卡顿、发热异常、内存抖动
- [ ] 横屏主场景布局无重叠、截断、错位
- [ ] 地图动效与反馈（聚焦、高亮、引导）无明显掉帧

## 5. UI 一致性检查

- [ ] 字体层级符合预期（标题/正文/标签/说明）
- [ ] 间距、圆角、卡片样式一致
- [ ] 按钮语义与状态一致（主按钮/次按钮/禁用态）
- [ ] 状态标签（进行中/完成/锁定等）颜色与文案一致

## 6. 建议回归组合

- [ ] 组合 A（快速 15 分钟）：Home -> Map -> 1 节课 -> Reward -> Chest -> Home
- [ ] 组合 B（功能 30 分钟）：Map 多岛切换 -> 不同题型各 1 题 -> Parent Gate -> Parent Summary
- [ ] 组合 C（专项 20 分钟）：Challenge 全链路（冲刺/评级/replay/retry）

## 7. 自动化验收基线

- [ ] `./gradlew.bat testDebugUnitTest`
- [ ] `./gradlew.bat assembleDebug`
- [ ] `./gradlew.bat connectedDebugAndroidTest`
- [ ] 如果 UTP 不稳定，改跑 `./scripts/run-focused-emulator-regression.ps1`

## 8. 缺陷记录建议

- [ ] 记录格式：`[模块][严重级别] 标题`
- [ ] 必填信息：设备/分辨率、构建版本、复现步骤、期望结果、实际结果、截图或录屏
- [ ] 标注是否阻塞发布：`Blocker / 非阻塞`
